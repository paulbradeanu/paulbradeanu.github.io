Tutorial I

Today I want to teach you how to script html too. First of all, you need to know the basics. To make a working html command, you have to put it into "< >". These must be before and after the text, but after it, put a "/" in front of the command to end it. I know it seems hard, but it's really easy. Let me show you some commands and I will explain you how to use them.

First of all, let's try to edit the size and the font of the text. I will use this sentence: I love the Internet.

The first command (with the "font size") is for the size of the text, and the secound command (with the "font face") is for the font of it. As you saw, for the "font size" I used "10", but you can use what number you want, and for the "font face" I used the "Arial" font, but there are a lot more fonts that you can use.

Tutorial II

Now, I want to show you how to change the look of you text with the "bold", "italic", and "underlined" add-ons.

As always, I will put there a picture with the script and I will explain you it.

For the first add-on (the "bold" one), I used the "strong" command, for the secound add-on (the "italic" one), I used the "em" command, and for the third add-on (the "underlined" one), I used the "u" command.

You can use the "bold" text for the titles or important things, the "italic" text for the quotes, and the "underlined" text for the marked things.
